# font-mapper-scripts

本目录包含 **「字体-韩文映射」** 的完整脚本套件，用于：
1. 比对两个字体文件的字形轮廓；
2. 找出“形状一致但 Unicode 不同”的韩文字符；
3. 生成映射表（f1→f2）与日志。

---

## 1 环境依赖
| 项目 | 最低版本 | 备注 |
| ---- | -------- | ---- |
| Python | 3.7 | 64 位，推荐 3.8+ |
| fontTools | 4.38.0 | `pip install -r requirements.txt` 自动装 |

---

## 2 使用说明
* 将原版字体（font目录下）和加密字体文件放置于同文件夹
* 下载脚本并修改内部路径于字体文件路径
* 运行脚本，得到韩文及其映射以及查漏日志各一份
